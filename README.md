## 📖 Overview

EVENITE is a comprehensive student event management platform designed to simplify the organization and participation of academic events such as workshops, hackathons, seminars, competitions, webinars, and cultural programs.

The platform provides dedicated dashboards for students, event hosts, and administrators while integrating AI-powered event promotion, analytics, gamification, certificate generation, and attendance management into a unified ecosystem.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/bensebastian21/evenite.git
cd evenite
```

### Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file in the server directory:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_cloudinary_api_key
CLOUDINARY_API_SECRET=your_cloudinary_api_secret
```

## ▶️ Running the Application
